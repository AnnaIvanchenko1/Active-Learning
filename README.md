# Active Learning: Сравнение стратегий выбора данных для классификации текстов

# Стек
Python, scikit-learn, NumPy, Matplotlib, SciPy, TF-IDF, MultinomialNB, KMeans, TruncatedSVD

# Описание проекта
Данный проект посвящен реализации и сравнительному анализу различных стратегий Active Learning для задачи классификации текстов на датасете 20 Newsgroups. Реализованы следующие подходы:

- Random — случайный выбор образцов для разметки (baseline)

- ClusterInitAL — инициализация центрами кластеров KMeans в LSA-пространстве

- QueryCommitteeAL — выбор по расхождению мнений комитета из 2 NB моделей

- LeastConfClusterAL — комбинация least confidence + кластеризация подвыборки

Проект учитывает ограниченный бюджет разметки (1M слов) и оптимизирует выбор наиболее информативных образцов.

# Цель проекта
Создать эффективные стратегии Active Learning для минимизации затрат на разметку при максимальной точности классификации текстов. Основные задачи:

- Сравнить качество различных AL стратегий по метрике F1-macro на тестовой выборке

- Оценить влияние начальной инициализации (random vs cluster centers) на итоговое качество

- Оптимизировать выбор образцов с учетом реальной стоимости разметки (длина текста в словах)

- Построить траектории обучения F1-score и затрат по бюджету

Бюджет: 1,000,000 слов (PRICE_THRESHOLD)

# Результаты и выводы
Реализованы и протестированы 4 стратегии Active Learning с учетом ограниченного бюджета разметки. Построены траектории F1-score и затрат для каждой стратегии.

# Ключевые выводы:

- Cluster инициализация значительно улучшает стартовое качество по сравнению с random

- Query-by-Committee эффективно выбирает информативные образцы по расхождению моделей

- LeastConf+Cluster комбинирует неуверенность модели с разнообразием кластеров

- Все стратегии укладываются в бюджет 1M слов, достигая различного качества F1-macro
